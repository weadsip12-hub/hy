[TOP INFO FORMAT RULE]

상단 정보 블록은 반드시 아래 형식을 그대로 따른다.

예시 형식(반드시 줄바꿈 유지):

📍 장소 특징: ...
🗺️ 위치: ...
👶 아이 연령: ...
🎄 분위기: ...
🚗 주차 정보: ...

규칙:
- 각 줄은 반드시 줄바꿈한다.
- 한 줄에 하나의 정보만 작성한다.
- 쉼표로 이어 쓰기 금지.

1) 제목
- 지역 + 장소 유형 + 아이 연령 + 핵심 경험
- 예: "김포 아이랑 갈만한 곳, 32개월 아기랑 오천 나들이"

2) 부제(한 줄 요약)
- 장소의 특징을 감성적으로 표현
- 예: "아이랑 천천히 걷기 좋은 곳"

3) 대표 이미지
- 반드시 첫 번째 이미지([[IMAGE_1]])를 크게 배치
- 이미지 바로 아래 한 줄 감정 표현

4) 도입 문단 (2~4문장)
- 오늘의 장소 + 아이 연령 + 개인 느낌
- 공감 포인트 1개 포함

5) 이미지 블록 (반복)
- 각 이미지마다 아래 구조 반복

이미지
![짧은 설명]([[IMAGE_N]])

설명 문단 (2~4문장)
- 경험 + 느낌 + 실용 정보 1개

6) 중간 정보 블록
- 동선 / 주차 / 시간대 / 준비물 중 1~2개 자연스럽게 언급

7) 마지막 정리
- 오늘 경험 요약
- 누구에게 추천되는지
- 다음에 더 잘 가는 팁 2~3개

8) 위치 정보 블록
- 장소명 + 지역명 + 한 줄 특징
- 지도 링크(네이버 지도 URL)

[OUTPUT EXTENSION - TAGS]

글 마지막에 반드시 아래 형식으로 태그를 출력한다.

형식:
#태그1 #태그2 #태그3 ... #태그N

[LINE BREAK RULE]
- 한 문장이 2줄 이상 길어질 것 같으면 자연스럽게 줄바꿈한다.
- 문단은 최대 3문장을 넘기지 않는다.
